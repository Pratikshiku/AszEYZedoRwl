### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Navicat`

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告
5. 流浪
6. 团队活动
7. 个人中心
8. 备忘录
9. 捐赠
10. 流浪收藏
11. 流浪订单
12. 团队活动收藏
13. 团队活动报名
14. 志愿者申请


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 用户管理
5. 流浪管理
6. 公告管理
7. 捐赠管理
8. 备忘录管理
9. 团队活动管理
10. 志愿者申请管理
11. 公告类型管理
12. 流浪类型管理
13. 团队活动类型管理
14. 用户类型管理
15. 轮播图管理
